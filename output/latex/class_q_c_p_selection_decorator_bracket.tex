\doxysubsection{Detailed Description}
A selection decorator which draws brackets around each selected data segment. 

Additionally to the regular highlighting of selected segments via color, fill and scatter style, this \doxylink{class_q_c_p_selection_decorator}{QCPSelection\+Decorator} subclass draws markers at the begin and end of each selected data segment of the plottable.

The shape of the markers can be controlled with \doxylink{class_q_c_p_selection_decorator_bracket_a04507697438f6ad8cc2aeea5422dcbe5_a04507697438f6ad8cc2aeea5422dcbe5}{set\+Bracket\+Style}, \doxylink{class_q_c_p_selection_decorator_bracket_a291b59cab98ce93a0a3c85963fe10f5e_a291b59cab98ce93a0a3c85963fe10f5e}{set\+Bracket\+Width} and \doxylink{class_q_c_p_selection_decorator_bracket_aed773ad737201cca40efc6fe451acad8_aed773ad737201cca40efc6fe451acad8}{set\+Bracket\+Height}. The color/fill can be controlled with \doxylink{class_q_c_p_selection_decorator_bracket_ac0e392a6097990f8aa978932a8fa05d6_ac0e392a6097990f8aa978932a8fa05d6}{set\+Bracket\+Pen} and \doxylink{class_q_c_p_selection_decorator_bracket_a2f4ea0bfb0ea980252b76dd349dd53aa_a2f4ea0bfb0ea980252b76dd349dd53aa}{set\+Bracket\+Brush}.

To introduce custom bracket styles, it is only necessary to sublcass \doxylink{class_q_c_p_selection_decorator_bracket}{QCPSelection\+Decorator\+Bracket} and reimplement \doxylink{class_q_c_p_selection_decorator_bracket_a57b65b5508d5bd0f27c9318f3d4646be_a57b65b5508d5bd0f27c9318f3d4646be}{draw\+Bracket}. The rest will be managed by the base class.
