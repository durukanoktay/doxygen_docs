The inner rect is the outer rect (\doxylink{class_q_c_p_layout_element_a2a32a12a6161c9dffbadeb9cc585510c_a2a32a12a6161c9dffbadeb9cc585510c}{outer\+Rect}, \doxylink{class_q_c_p_layout_element_a38975ea13e36de8e53391ce41d94bc0f_a38975ea13e36de8e53391ce41d94bc0f}{set\+Outer\+Rect}) shrinked by the margins (\doxylink{class_q_c_p_layout_element_a8f450b1f3f992ad576fce2c63d8b79cf_a8f450b1f3f992ad576fce2c63d8b79cf}{set\+Margins}, \doxylink{class_q_c_p_layout_element_accfda49994e3e6d51ed14504abf9d27d_accfda49994e3e6d51ed14504abf9d27d}{set\+Auto\+Margins}).

This event gets called when the user turns the mouse scroll wheel while the cursor is over the layerable. Whether a cursor is over the layerable is decided by a preceding call to \doxylink{class_q_c_p_abstract_legend_item_a206879d493749c3fb1494850bff40e7e_a206879d493749c3fb1494850bff40e7e}{select\+Test}.

The current pixel position of the cursor on the \doxylink{class_q_custom_plot}{QCustom\+Plot} widget is accessible via {\ttfamily event-\/\texorpdfstring{$>$}{>}pos()}.

The {\ttfamily event-\/\texorpdfstring{$>$}{>}angle\+Delta()} indicates how far the mouse wheel was turned, which is usually +/-\/ 120 for single rotation steps. However, if the mouse wheel is turned rapidly, multiple steps may accumulate to one event, making the delta larger. On the other hand, if the wheel has very smooth steps or none at all, the delta may be smaller.

The default implementation does nothing.
