\doxysubsection{Detailed Description}
Describes a data range given by begin and end index. 

\doxylink{class_q_c_p_data_range}{QCPData\+Range} holds two integers describing the begin (\doxylink{class_q_c_p_data_range_a54ff59048e01e46ac4aefafc844626c6_a54ff59048e01e46ac4aefafc844626c6}{set\+Begin}) and end (\doxylink{class_q_c_p_data_range_a277f1a9eafe70b9184d9c00b641ae5de_a277f1a9eafe70b9184d9c00b641ae5de}{set\+End}) index of a contiguous set of data points. The {\itshape end} index corresponds to the data point just after the last data point of the data range, like in standard iterators.

Data Ranges are not bound to a certain plottable, thus they can be freely exchanged, created and modified. If a non-\/contiguous data set shall be described, the class \doxylink{class_q_c_p_data_selection}{QCPData\+Selection} is used, which holds and manages multiple instances of \doxylink{class_q_c_p_data_range}{QCPData\+Range}. In most situations, \doxylink{class_q_c_p_data_selection}{QCPData\+Selection} is thus used.

Both \doxylink{class_q_c_p_data_range}{QCPData\+Range} and \doxylink{class_q_c_p_data_selection}{QCPData\+Selection} offer convenience methods to work with them, e.\+g. \doxylink{class_q_c_p_data_range_a93529421d12fdd3a8bdb2b8061936352_a93529421d12fdd3a8bdb2b8061936352}{bounded}, \doxylink{class_q_c_p_data_range_a36c8ad8acf177ffeb0a72c8d73030844_a36c8ad8acf177ffeb0a72c8d73030844}{expanded}, \doxylink{class_q_c_p_data_range_a8a1c437f9beffd55621f48f961d7f679_a8a1c437f9beffd55621f48f961d7f679}{intersects}, \doxylink{class_q_c_p_data_range_a84e1e03129dd52528efb4bac18d30183_a84e1e03129dd52528efb4bac18d30183}{intersection}, \doxylink{class_q_c_p_data_range_a279ed36602b39607699dc5652bcaf813_a279ed36602b39607699dc5652bcaf813}{adjusted}, \doxylink{class_q_c_p_data_range_a13ca398776374a3160aa85433718b812_a13ca398776374a3160aa85433718b812}{contains}. Further, addition and subtraction operators (defined in \doxylink{class_q_c_p_data_selection}{QCPData\+Selection}) can be used to join/subtract data ranges and data selections (or mixtures), to retrieve a corresponding \doxylink{class_q_c_p_data_selection}{QCPData\+Selection}.

QCustom\+Plot\textquotesingle{}s data selection mechanism is based on \doxylink{class_q_c_p_data_selection}{QCPData\+Selection} and \doxylink{class_q_c_p_data_range}{QCPData\+Range}.

\begin{DoxyNote}{Note}
Do not confuse \doxylink{class_q_c_p_data_range}{QCPData\+Range} with \doxylink{class_q_c_p_range}{QCPRange}. A \doxylink{class_q_c_p_range}{QCPRange} describes an interval in floating point plot coordinates, e.\+g. the current axis range. 
\end{DoxyNote}

bool QCPData\+Range\+::is\+Valid (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [inline]}}

Returns whether this range is valid. A valid range has a begin index greater or equal to 0, and an end index greater or equal to the begin index.

\begin{DoxyNote}{Note}
Invalid ranges should be avoided and are never the result of any of \doxylink{class_q_custom_plot}{QCustom\+Plot}\textquotesingle{}s methods (unless they are themselves fed with invalid ranges). Do not pass invalid ranges to \doxylink{class_q_custom_plot}{QCustom\+Plot}\textquotesingle{}s methods. The invalid range is not inherently prevented in \doxylink{class_q_c_p_data_range}{QCPData\+Range}, to allow temporary invalid begin/end values while manipulating the range. An invalid range is not necessarily empty (\doxylink{class_q_c_p_data_range_ac32e53fc05d6cd2eac96b96a7265d3b8_ac32e53fc05d6cd2eac96b96a7265d3b8}{is\+Empty}), since its \doxylink{class_q_c_p_data_range_a1e7836058f755c6ab9f11996477b7150_a1e7836058f755c6ab9f11996477b7150}{length} can be negative and thus non-\/zero. 
\end{DoxyNote}
