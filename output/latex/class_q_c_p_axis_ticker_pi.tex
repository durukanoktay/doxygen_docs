\doxysubsection{Detailed Description}
Specialized axis ticker to display ticks in units of an arbitrary constant, for example pi. 



This \doxylink{class_q_c_p_axis_ticker}{QCPAxis\+Ticker} subclass generates ticks that are expressed with respect to a given symbolic constant with a numerical value specified with \doxylink{class_q_c_p_axis_ticker_pi_a36ce0651d2ec92edd36feac1619c2468_a36ce0651d2ec92edd36feac1619c2468}{set\+Pi\+Value} and an appearance in the tick labels specified with \doxylink{class_q_c_p_axis_ticker_pi_acfdcd4758a393bde4be12a50fb2017b5_acfdcd4758a393bde4be12a50fb2017b5}{set\+Pi\+Symbol}.

Ticks may be generated at fractions of the symbolic constant. How these fractions appear in the tick label can be configured with \doxylink{class_q_c_p_axis_ticker_pi_a760c8af6ca68178e607556c4e5049d71_a760c8af6ca68178e607556c4e5049d71}{set\+Fraction\+Style}.
