\doxysubsection{Detailed Description}
Holds the data of one single data point for \doxylink{class_q_c_p_graph}{QCPGraph}. 

The stored data is\+: \begin{DoxyItemize}
\item {\itshape key\+:} coordinate on the key axis of this data point (this is the {\itshape main\+Key} and the {\itshape sort\+Key}) \item {\itshape value\+:} coordinate on the value axis of this data point (this is the {\itshape main\+Value})\end{DoxyItemize}
The container for storing multiple data points is \doxylink{qcustomplot_8h_a9acaec6c7811dbc3f9ca07b6631f0442_a9acaec6c7811dbc3f9ca07b6631f0442}{QCPGraph\+Data\+Container}. It is a typedef for \doxylink{class_q_c_p_data_container}{QCPData\+Container} with \doxylink{class_q_c_p_graph_data}{QCPGraph\+Data} as the Data\+Type template parameter. See the documentation there for an explanation regarding the data type\textquotesingle{}s generic methods.
