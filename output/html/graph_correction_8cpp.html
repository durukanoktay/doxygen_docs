Detailed Description

Corrects initial oscillation errors in CBR test graphs.

This function is written to correct the errors caused by oscillations at the beginning of the test, which occur due to the test table or other reasons during CBR tests.

The function's tasks are as follows:

  • Determine when the load exactly starts to apply on the sample.
  • Return the displacement value at the moment the load starts to apply.

Usage:

  • The correction value returned by this function is used to recalculate the test results, graphs, and other calculations, thereby eliminating errors.
